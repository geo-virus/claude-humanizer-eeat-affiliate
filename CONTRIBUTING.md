# Contributing to Claude Humanizer — E-E-A-T Affiliate Edition

Thanks for helping make affiliate content less robotic and more trustworthy.

---

## How to Contribute

### 1. Report AI Patterns We Miss

Found a robotic pattern this skill doesn't catch? Open an issue with:
- **Pattern name**
- **Example sentence** (AI-generated)
- **Why it reads as AI**
- **Suggested rewrite**

### 2. Submit Niche-Specific E-E-A-T Examples

Different niches need different trust signals:
- **Finance:** Regulatory disclaimers, SEC references, risk warnings
- **Health:** Medical reviewer credits, "not medical advice" clarity
- **Software:** Screenshots described, version numbers, OS compatibility
- **Physical products:** Weight, dimensions, unboxing experience, warranty details

Submit these as additions to the E-E-A-T injection section.

### 3. A/B Test Data

If you test humanized vs. raw AI content:
- Time on page
- Bounce rate
- Affiliate CTR
- Ranking movement

We'd love to include anonymized case studies.

### 4. Skill File Improvements

- Fork the repo
- Edit `skills/humanizer-eeat-affiliate/SKILL.md`
- Test with 3+ real articles
- Submit a PR with before/after samples

---

## Style Guide for SKILL.md Edits

- Keep pattern IDs sequential (P21, P22, etc.)
- Every pattern needs: ID, name, AI example, rewrite rule
- Keep lines under 100 characters where possible
- Use markdown tables for structured data
- Test your changes in Claude Code before PRing

---

## Code of Conduct

- Be constructive. We're all fighting the same robot uprising.
- Share real data when possible. Opinions are fine, but metrics are better.
- Respect that different niches have different compliance requirements.
